## 3. Projenin Amacı ve Kısa Açıklaması
Birleşmiş Milletler Çevre Programı (UNEP) verilerine göre, küresel olarak üretilen gıdaların yaklaşık %40'ı tüketilmeden çöpe gitmektedir. Organik atıkların çürümesiyle atmosfere salınan metan ($CH_4$) gazı ise küresel ısınmayı tetikleyen en büyük faktörlerden biridir.

**AuraCook**, evsel gıda yönetimini optimize etmek, sürdürülebilir mutfak alışkanlıkları kazandırmak ve gıda israfının karbon ayak izini azaltmak amacıyla geliştirilmiş yapay zekâ destekli ve oyunlaştırma (gamification) odaklı bir mobil uygulamadır.

**Temel Özellikler:**
*   **Akıllı Dolap Yönetimi:** Dolaptaki gıdaların takibini kolaylaştıran, emoji chipli reaktif envanter sistemi.
*   **Üretken Yapay Zekâ Şefi:** Dolapta bulunan malzemeleri analiz ederek Groq Cloud (Llama 3) aracılığıyla sıfır atık odaklı adım adım tarifler üreten yapay zekâ motoru.
*   **Eller Serbest (Hands-Free) Pişirme Modu:** Mutfakta elleri kirli olan kullanıcılar için ses tanıma (Speech-to-Text) ve ses sentezleme (Text-to-Speech) entegrasyonuyla çalışan sesli tarif asistanı.
*   **Mutfak Auram & Ekolojik Etki Raporu:** Çöpe atılmaktan kurtarılan gıdalara göre karbondioksit ($CO_2$) tasarrufunu IPCC ve FAO standartlarına göre hesaplayan ve rozetlerle kullanıcıyı ödüllendiren oyunlaştırma sistemi.
*   **Sosyal Medya Akışı & Topluluk:** Çevre bilincine sahip şeflerin tariflerini paylaştığı, beğeni, yorum ve liderlik tablosu (leaderboard) içeren topluluk platformu.

## 4. Kullanılan Teknolojiler / Kütüphaneler
Uygulama, kurumsal standartlarda (Enterprise-grade) modern mobil yazılım mimarisiyle sıfırdan geliştirilmiştir:

### A. Çekirdek Teknolojiler
*   **Flutter (v3.22+):** Çapraz platform mobil uygulama geliştirme çerçevesi.
*   **Dart (v3.4+):** Güçlü tipli, reaktif programlama dili.

### B. Durum Yönetimi ve Veri Tabanı
*   **Flutter Riverpod (v3.3.1) & Riverpod Generator:** Kod üretimi (`build_runner`) destekli, tür güvenli, reaktif state management altyapısı.
*   **Hive NoSQL Database (v2.2.3) & Hive Flutter:** Çevrimdışı öncelikli (**Offline-First**) mimariyi destekleyen yerel NoSQL veritabanı. Gıdalar, tarif geçmişi ve favoriler cihaz hafızasında güvenle cache'lenir.

### C. Yapay Zekâ ve Dış API Servisleri
*   **Groq Cloud API (Llama-3-70b-8192):** Yüksek hızlı yapay zekâ tarif üreticisi ve İngilizce-Türkçe akıllı yiyecek çeviri motoru.
*   **FatSecret API (OAuth 2.0):** Küresel besin veritabanı entegrasyonu. Malzemelerin kalorilerini, porsiyonlarını ve makro değerlerini çeker.
*   **YouTube Data API v3:** Reels sekmesi için sürdürülebilir mutfak ve sıfır atık içerikli dikey videoları çeker.
*   **Wikipedia Image API:** Görseli eksik olan gıdalar için arka planda görsel arar ve önbelleğe alır.

### D. Önemli Kütüphaneler (Packages)
*   `speech_to_text`: Eller serbest pişirme modu için sesli komut algılama.
*   `flutter_tts`: Tarif adımlarının sesli okunması için ses sentezleme motoru.
*   `cached_network_image`: Görselleri cihazda önbelleğe alarak internet ve batarya tasarrufu sağlar.
*   `flutter_animate`: Premium staggered arayüz ve onboarding geçiş animasyonları.

│   ├── home/            # "Aura Dolabım" envanter yönetim ve malzeme ekleme arayüzü
│   ├── recipes/         # Yapay zekâ şefi, tarif detayları ve sesli asistan ekranları

## 6. Kurulum Adımları
Projeyi bilgisayarınızda derlemek ve çalıştırmak için aşağıdaki adımları sırasıyla uygulayınız:

### Prerequisities (Gereksinimler)
*   Bilgisayarınızda **Flutter SDK** yüklü olmalıdır. (Kurulum rehberi için: [flutter.dev](https://docs.flutter.dev/get-started/install))
*   Bir mobil cihaz veya emülatör hazır bulunmalıdır.

### Adım 1: Depoyu Klonlayın
```bash
git clone https://github.com/emirhandalgiran/Yapay-Zeka-Destekli-Mutfak-ve-Diyet-Asistani.git
cd Yapay-Zeka-Destekli-Mutfak-ve-Diyet-Asistani
```

### Adım 2: Bağımlılıkları Yükleyin
Gerekli tüm paketleri indirmek için terminalde şu komutu çalıştırın:
```bash
flutter pub get
```

### Adım 3: Ortam Değişkenlerini Tanımlayın (.env)
Proje güvenliği gereği API anahtarları kaynak kodda gizlenmiştir.
1. Proje ana dizinindeki `.env.example` dosyasının bir kopyasını oluşturup adını `.env` yapın.
2. Dosya içerisindeki `GROQ_API_KEY`, `FATSECRET_CLIENT_ID`, `FATSECRET_CLIENT_SECRET` gibi ilgili alanlara kendi API anahtarlarınızı yazın.

## 7. Çalıştırma / Kullanım Talimatları

### Geliştirici Modunda Çalıştırma (Run)
Cihazınız bağlıyken terminale şu komutu yazarak projeyi başlatabilirsiniz:
```bash
flutter run
```

### Kod Üretimi (Riverpod & Hive için)
Model veya provider katmanlarında değişiklik yapıldığında kod üreticileri çalıştırmak için:
```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

### Temel Kullanım Senaryoları
1.  **Dolap Ekranı:** "Aura Dolabım" sayfasından hızlı emoji chiplerine tıklayarak veya arama çubuğunu kullanarak elinizdeki malzemeleri envanterinize ekleyin.
2.  **Yapay Zekâ Şefi:** Eklediğiniz malzemelerden birini seçerek detay sayfasına gidin ve "Tarife Dönüştür" butonuna basın. Groq Llama 3 saniyeler içinde sıfır atık ipuçları barındıran bir tarif oluşturacaktır.
3.  **Eller Serbest Modu:** Tarif adımları açıldığında mikrofona erişim izni vererek ellerinizi kullanmadan *"Sonraki"*, *"Önceki"* veya *"Tekrar Et"* komutlarıyla asistanı yönetin. Cihaz adımları size sesli okuyacaktır.
4.  **Mutfak Auram:** Tarif uyguladıktan sonra profildeki karbon tasarrufunuzu ve kilidi açılan başarı rozetlerinizi inceleyin.
